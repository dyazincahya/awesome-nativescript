# NativeScript Examples Collection

Welcome to the NativeScript examples collection! This directory presents various implementations and code snippets showcasing NativeScript features with diverse framework flavors.

Each example is designed to be self-contained and easy to understand, helping you explore NativeScript's capabilities. Find the examples relevant to your needs in the table below:

- [Core](core) - Example using pure NativeScript with TypeScript/JavaScript.
- [Angular](angular) - Example using NativeScript with Angular.
- [Vue](vue) - Example using NativeScript with Vue.
- [React](react) - Example using NativeScript with React.
- [Solid](solid) - Example using NativeScript with Solid.
- [Svelte](svelte) - Example using NativeScript with Svelte.
